Pythonの静的サイトジェネレーターPelicanでGitHubにブログを開設する 1

Posted on 日 11 3月 2018 in python • Tagged with pelican, python, github

はじめに

たまに書き留める備忘録的なブログサイトを作りたいなと思ったものの、WordPressを使うとなると無料で運用出来ない。
で、GitHub Pages（GitHub Pagesがどういうものなのか知らなかったけど）にPelicanを使ってブログ作る方法というのを知ったので作ってみようと。
最近HerokuにTwitter Botを作ってGitに触れたからGitHubに興味を持ったというのもあります。

Githubに登録

Gitがインストールされているという前提で進めていきます。
まずはHerokuでアカウントを作成 登録したメールアドレスにメールが届くのでリンクをクリックしてverifyする。

レポジトリを作成

Repository nameにユーザーネーム.github.ioと打ち込んでInitialize this repository with a READMEにチェックを入れ、Create Repositoryをクリックして新規作成。

ローカルに環境を構築

pelican markdownとghp-importをインストール

pmake serve でサーバーが起動しなかったので、outputフォルダに移動して,py -m http.server、もしくはpy -m pelican.serverで起動 …
